简介

本人性格热情开朗，待人友好，为人诚实谦虚。工作勤奋，认真负责，能吃苦耐劳，尽职尽责，有耐心。具有亲和力，平易近人，善于与人沟通。在大学期间学习认真，对知识的掌握情况如下：

C++

学习了C++的相关知识，虽然基础较为薄弱，但是以后会继续努力。

数字电路and模拟电路

学习了硬件电路的基础知识，成功磨灭了我对硬件的兴趣。

Java

认真的学习了Java相关知识，自认为掌握的还不错，以后还会继续学习。

通信原理

通信原理使我对于通信的相关原理有了初步的掌握，并未以后的工作和学习奠定了坚实的基础。

该门课程帮助我对信源、信道的相关原理有了初步的认识，对香农印象深刻。
